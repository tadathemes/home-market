Home Market Theme Installation

It's too easy to install Home Market theme. Please log in to your Shopify store and go to Store Online > Themes. From there, click on Upload a Theme button and choose the homemarket-xxx-yyy.zip (xxx is template name such as main, yyy is version update) file that included in purchase package download.

If your site is currently active and using a different theme, please click on Publish theme button of Home Market package have just uploaded.

Horizontal Mega Menu

Please log in to your Shopify store and go to Store Online > Themes > Customize theme > Mega Menu



For example, we have Main Menu with links: Home, Collections, Shop, Today's Deals, Blog, LookBook
We need the Shop link is Mega menu with drop-down content is Link List
  1. In Mega Menu for Link List layout setting, set "Mega Menu Title" is Shop. That mean input the name of link in Main Menu that you want it will be a Mega Menu.

  2. For each column, you have 4 options:

    • Sample Text (HTML): insert content for 1st column
    • Link List 1: set the link list that you want show in 2nd column
    • Link List 2: set the link list that you want show in 3rd column
    • Link List 3: set the link list that you want show in 4th column
Note: You need create the Link list have links you want first. Then you can select it in above step.


You can set the Special label for these menu

Vertical Mega Menu (All Collections)

Please log in to your Shopify store and go to Store Online > Themes > Customize theme > All Collections Listing

There are are many options:

  • Heading text: input the name of title block
  • Select Collections link list: With this method you can sort the collections as you want. Default Shopify sort by aphabet. You need create Link list have collections that you want show in this block first.
  • Show icon before the Collection name?: active icon in the left of the text. You need upload these icons into Assets folder with the name is collection_handle_name.png
  • Show number product after the Collection name?: active show number products for each collection


You can set the dropdown content for these menu

SideBar Blocks

Please log in to your Shopify store and go to Store Online > Themes > Customize theme > Product Page



There are many options to control the SideBar blocks:
  • All Collections Block will show all Collections in your site.
  • Advanced AJAX Tag filter will show Shop by block
  • Sample Block this is sample block with static text. You can insert any information that you want.
  • Suggested Products Block show products form selected collection that you want suggest to customer.
  • Recent Viewed Products will show recent viewed/accessed product
  • Slidebar Banner show sidebar banner
  • Product Vendors Block show all vendors or your products. Default is turn off.
  • Product Types Block show all type or your products. Default is turn off.
  • Link List in Slidebar show link list that you want. Default is turn off.

Advanced AJAX Tag filter (Shop By block)

This is tags product sort filter
To show it you need:

1. Add tags to your products

You must add the tags you wish to filter your products on.
For example if you want to filter by Color then you must add the color tags to each product (e.g. edit all products and add the tag “red” to them if the product is red).

2. Insert tag value to the Theme Settings

Please log in to your Shopify store and go to Store Online > Themes > Customize theme > Sidebar > Advanced AJAX Tag filter and input options
